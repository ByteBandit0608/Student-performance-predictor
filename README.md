# Student-performance-predictor
This project is a simple machine learning model that predicts whether a student is likely to pass or fail based on the number of hours they study. The goal of the project was to understand the basics of machine learning workflows such as loading a dataset, training a model, and making predictions.
# Student Performance Predictor

This project uses a simple machine learning model to predict whether a student will pass or fail based on study hours.

Technologies
- Python
- Pandas
- Scikit-learn

Dataset
The dataset contains study hours and whether the student passed or failed.

How to Run

1. Install dependencies
pip install -r requirements.txt

2. Run the model
python train_model.py
